# SJ GILL Mobile Car Detailing Website

Marketing site for **SJ GILL Mobile Car Detailing** built with Vite, React, TypeScript and Tailwind CSS.

## How can I edit this code?

Clone and work locally in your preferred editor:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

You can deploy this project using any static hosting provider that supports Vite builds, for example:

- Netlify
- Vercel
- GitHub Pages

Typical steps:

```sh
npm run build
```

Then configure your hosting provider to serve the `dist` directory.
